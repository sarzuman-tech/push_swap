# Libft - My Standard C Library

## 📝 Description
**Libft** — это индивидуальный проект в школе 42, целью которого является создание собственной библиотеки стандартных функций языка C. В ходе работы были переписаны функции из стандартных библиотек `libc`, а также реализованы дополнительные утилиты для работы со строками, памятью и связными списками.

Основная задача проекта — глубокое понимание того, как работают стандартные функции «под капотом», освоение управления памятью (динамическое выделение) и работа с указателями. Эта библиотека станет базой для всех последующих проектов на C в рамках обучения.

## 🛠️ Instructions

### Compilation
Для компиляции проекта используется `Makefile`. Все файлы должны находиться в корневой директории.

| Команда | Результат |
| :--- | :--- |
| `make` | Сборка основной части библиотеки `libft.a` |
| `make bonus` | Добавление функций для работы со списками |
| `make clean` | Удаление объектных файлов (`.o`) |
| `make fclean` | Полная очистка: удаляет `.o` и файл `libft.a` |
| `make re` | Пересборка библиотеки с нуля |

## 🛠️ Функционал библиотеки

### 1. Функции из `<ctype.h>`
Проверка и преобразование символов:
- `ft_isalpha` — проверка на букву.
- `ft_isdigit` — проверка на цифру.
- `ft_isalnum` — проверка на букву или цифру.
- `ft_isascii` — проверка на таблицу ASCII.
- `ft_isprint` — проверка на печатаемый символ.
- `ft_toupper` / `ft_tolower` — смена регистра.

### 2. Функции из `<string.h>` и `<strings.h>`
Работа со строками и памятью:
- `ft_strlen` — длина строки.
- `ft_memset` — заполнение памяти байтом.
- `ft_bzero` — зануление памяти.
- `ft_memcpy` / `ft_memmove` — копирование областей памяти.
- `ft_strlcpy` / `ft_strlcat` — безопасное копирование и конкатенация строк.
- `ft_strchr` / `ft_strrchr` — поиск символа в строке.
- `ft_strncmp` — сравнение строк.
- `ft_memchr` / `ft_memcmp` — поиск и сравнение в блоках памяти.
- `ft_strnstr` — поиск подстроки в строке.

### 3. Функции из `<stdlib.h>`
- `ft_atoi` — конвертация строки в целое число.
- `ft_calloc` — выделение памяти с занулением.
- `ft_strdup` — создание дубликата строки.

### 4. Дополнительные функции
Специфические функции, отсутствующие в стандарте C или имеющие другой формат:
- `ft_substr` — создание подстроки.
- `ft_strjoin` — объединение двух строк в новую.
- `ft_strtrim` — обрезка символов с начала и конца строки.
- `ft_split` — разбиение строки на массив строк по разделителю.
- `ft_itoa` — конвертация числа в строку.
- `ft_strmapi` / `ft_striteri` — итерация по строке с применением функции.
- `ft_putchar_fd` / `ft_putstr_fd` / `ft_putendl_fd` / `ft_putnbr_fd` — вывод данных в дескриптор файла.

### 5. Bonus часть (Linked Lists)
Функции для работы с односвязными списками:
- `ft_lstnew`, `ft_lstadd_front`, `ft_lstsize`, `ft_lstlast`, `ft_lstadd_back`, `ft_lstdelone`, `ft_lstclear`, `ft_lstiter`, `ft_lstmap`.
